## 一、下载与安装

### iPhone / iPad

1. Safari 打开：**https://jym1991.github.io/tomato-clock/**
2. 底部点「分享」按钮（方框+箭头）
3. 选择「**添加到主屏幕**」
4. 命名「番茄时钟」→ 添加
5. 桌面出现番茄图标，点击即用

### Mac

1. 获取 `番茄时钟_v6_macOS.dmg`
2. 双击挂载 → 拖入 Applications
3. 首次打开：右键 → 打开

### Windows / Linux

1. 获取对应平台的文件夹
2. 双击 `启动番茄时钟.bat`（Win）或 `./启动番茄时钟.sh`（Linux）
3. 在 Chrome 中打开后 → 地址栏右侧安装按钮 → 创建桌面应用

## 二、基本使用

### 番茄计时

| 操作 | 方式 |
|------|------|
| 开始/暂停 | 点中间 **▶** 按钮 |
| 重置 | 点左侧 **↺** 按钮 |
| 跳过 | 点右侧 **⏭** 按钮 |
| 调整时长 | 点上方 25分/45分/60分 |
| 调整休息 | 点上方 休5/休10 |

### 专注结束

每次 25 分钟专注结束后：
1. 弹窗「请休息一会儿」+ 名人名言 + 5 分钟倒计时
2. 倒计时结束 → 点「休息完成，继续工作」
3. 弹出微习惯打卡（起身走动 / 正向活动 / 正念呼吸 / 表情管理）
4. 打卡完成 → 自动开始下一轮

### iOS 特性

- **打开即开始**：进入 App 自动开始计时
- **切出暂停**：切换 App 自动暂停
- **回来继续**：切回来自动恢复

### Mac 特性

- **菜单栏进度环**：顶部菜单栏显示番茄进度
- **自动检测**：检测你在用工作 App 时自动开始
- **娱乐暂停**：打开游戏/视频 App 自动暂停
- **最小化弹窗**：窗口关闭到托盘也会弹出休息提醒

## 三、健康数据

### 在 iPhone 上导入

**方式一：导入 XML（直接）**

1. 点击「健康」标签 → 点「**导入 XML**」
2. 在文件选择器中找到你的 Apple Health 导出文件（`导出.xml`）
3. 选择文件 → 自动解析 → 显示进度
4. 解析完成 → 29 项健康指标自动展示

> 注意：如果 XML 文件较大（>500MB），建议用 Mac 版导入后导出 JSON。

**方式二：加载 JSON（更快）**

1. 在 Mac 版中：右键菜单栏图标 →「导出健康数据缓存」
2. 将导出的 `daily_health.json` 传到 iPhone
3. iPhone 上点击「加载 JSON」→ 选择该文件

### 在 Mac 上导入

1. 确保 Apple Health 导出文件位于：
   `~/Documents/apple_health_export/导出.xml`
2. 点击「健康」→「**导入 XML**」
3. 自动解析（618 万条记录约 2-3 分钟）
4. 之后每次启动自动读取数据库，无需重新导入

### 健康数据导出（Mac → 其他设备）

1. Mac 版右键菜单栏 →「导出健康数据缓存」
2. 保存 JSON 文件
3. 传到 iPhone / Windows / Linux → 用「加载 JSON」导入

## 四、界面说明

### 首页（计时）

```
CPU 23%  |  内存 58%  |  ⚖ 82.5kg  |  🍅 今日 3 个
────────────────────────────────────────────
         ╭──────╮
         │ 23:15 │   ← 倒计时
         │ 专注   │
         ╰──────╯
       ↺    ▶    ⏭
    25分 45分 60分 | 休5 休10
```

- CPU/内存：过去 30 分钟均值
- 体重：过去 30 天均值（来自健康数据）
- 今日番茄：当天完成的番茄数，凌晨自动归零

### 统计页

- 今日 / 本周 / 总计番茄数
- SVG 趋势图表（支持 7天/30天/90天/年/全部）
- 微习惯打卡分布图

### 健康页

- 6 项核心指标：体重、血压、VO2Max、步行距离、心率变异性、静息心率
- 29 项完整指标（分类浏览）
- 科研级 SVG 图表 + 健康基准范围带
- 年龄参考范围（35 岁男性）

## 六、常见问题

**Q: 番茄数第二天没有归零？**
A: 已修复，本地时间凌晨 0:00 自动归零。

**Q: iPhone 上如何删除？**
A: 长按桌面图标 → 删除书签。

**Q: 数据会丢失吗？**
A: 番茄记录存在浏览器 localStorage，清除浏览器数据会丢失。建议定期用 Mac 版备份。

**Q: 健康数据基准是什么？**
A: 基于 35 岁男性的医学参考范围。可在技术文档中查看完整基准表。
